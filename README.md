# PopocatepetlVolcano
Master in Data Science research stay in Popocatepetl volcano

# Setup

## Python environment

To create the python environment, you can run the following command:

# python 3.12.3 and install the requirements
conda create -n estancia_popocatepetl python= 3.12.3
conda activate estancia_popocatepetl
pip install -r requirements.txt
